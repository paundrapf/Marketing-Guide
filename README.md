Etika Penting

Panduan ini include taktik "gelap"—psych manipulation. Pakai dengan tanggung jawab.

Yang Boleh:
- Guide keputusan yang beneficial untuk customer
- Transparansi, jangan menipu
- Respect privacy
- Kasih value real, bukan cuma extract

Yang Jangan:
- Bohong (walau convert tinggi)
- Exploit vulnerable audience
- Spam tanpa henti
- Fake scarcity/urgency
- Hard-sell produk jelek

Marketing = Persuasi, bukan manipulasi. Tau bedanya.
